# social-vet

This is the Spring Boot project. It uses PostgreSQL as Database and it includes CRUD operations.

It serves services with RestAPI.

You can make requests for following services;
- /animals
- /saveanimal
- /deleteanimal
- /searchbyanimalname
- /searchbypetsitterusername
- /petsitters
- /savepetsitter
- /deletepetsitter
- /petsitterbyusername
- /animaltypes
- /saveanimaltype
